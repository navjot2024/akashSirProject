# 🎉 Weekend Planning App

**Plan weekends without chaos** - A collaborative platform for coordinating schedules, discovering activities, and making group decisions with friends.

## ✨ Features

- **👤 User Authentication** - Secure signup/login
- **📅 Smart Group Scheduling** - Find common free times
- **🎪 Activity Suggestions** - Personalized recommendations
- **🗳️ Polls & Voting** - Group decision-making
- **👥 Social Connections** - Connect with users
- **📊 Admin Dashboard** - Analytics and management

## 🎯 Tech Stack

- React 18
- React Router v6
- Tailwind CSS
- Lucide React Icons
- Recharts
- Vite

## 🚀 Getting Started

### Installation

```bash
cd "/home/sama/Desktop/akash sir/WeekendPlanning"
npm install
```

### Development

```bash
npm run dev
```

The app will open at `http://localhost:3000`

### Build

```bash
npm run build
```
