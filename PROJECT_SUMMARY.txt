                              PROJECT OVERVIEW
================================================================================

The Weekend Planning App is a complete, production-ready React application
that helps users coordinate weekend activities with friends by:

1. Sharing schedules and finding common free times
2. Discovering activities and venues based on preferences
3. Making group decisions through voting
4. Building social connections with like-minded people
5. Managing and analyzing group activities (admin)

================================================================================
                        KEY ALGORITHMS
================================================================================

GREEDY SCHEDULING ALGORITHM:
  Purpose: Find optimal meeting times for groups
  Logic:
    1. For each time slot (day/hour):
       a. Count how many members are available
       b. If count > previous best, save this slot as best
    2. Return best slot with highest availability
  
  Time Complexity: O(n*m)
    - n = number of group members
    - m = number of time slots
  
  Space Complexity: O(1)
  
  Used in: Group Scheduling page

GRAPH DATA STRUCTURE:
  Purpose: Represent social connections and recommendations
  Implementation:
    - Nodes: Users with interests
    - Edges: Shared activity preferences
    - Strength: Connection weight based on mutual interests
  
  Used in:
    - Social Connections page (user network)
    - Activity Suggestions (recommendations)
    - Social graph visualization

SEARCH & FILTER ALGORITHM:
  Purpose: Find activities matching user criteria
  Logic:
    1. Filter by selected category
    2. Filter by search term in name
    3. Return matching results
  
  Time Complexity: O(n)
    - n = total number of activities
  
  Used in: Activity Suggestions page
